    PHASER.IO



    Phaser es un marco de juego 2D de software gratuito para hacer juegos HTML5 para computadoras de escritorio y móviles. Fue creado por Photon Storm. [2] Phaser usa un renderizador Canvas y WebGL internamente y puede intercambiarse automáticamente de acuerdo con el soporte del navegador. Esto permite una representación rápida en equipos de escritorio y dispositivos móviles. Utiliza la biblioteca Pixi.js para renderizar. Los juegos se pueden compilar para iOS, Android y aplicaciones de escritorio nativas a través de herramientas de terceros como Apache Cordova.
